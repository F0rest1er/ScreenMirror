# ScreenMirror
Бесплатный проект для трансляции экрана macOS на другие устройства (например, Android планшеты) через локальную сеть. Приложение предоставляет удобный графический интерфейс с возможностью выбора экрана для трансляции и автоматически генерирует QR-код для быстрого подключения с мобильного устройства.

## Использование
> **Важно:** Для трансляции второго экрана (в качестве расширения рабочего стола) понадобится бесплатная программа **BetterDisplay**. В ней необходимо создать виртуальный экран, который затем появится в списке доступных мониторов для трансляции.
1. Запустите приложение на Mac.
2. Откроется окно приложения с QR-кодом.
3. Если у вас несколько мониторов, выберите нужный экран из списка под QR-кодом.
4. Отсканируйте камерой вашего планшета или смартфона QR-код.
5. Откроется веб-страница в браузере мобильного устройства, где начнется трансляция выбранного экрана.

## Особенности
- **Трансляция в реальном времени**: использует захват экрана и стриминг через HTTP.
- **Поддержка нескольких мониторов**: возможность выбора конкретного монитора или виртуального экрана (например, созданного через BetterDisplay).
- **Графический интерфейс**: приложение запускается в отдельном нативном окне macOS благодаря использованию `webview`.
- **Удобное подключение**: локальный IP-адрес определяется автоматически, генерируется QR-код, который при сканировании сразу открывает браузер телефона/планшета на нужной странице.
- **Отсутствие внешних зависимостей для просмотра**: на устройстве-получателе требуется только обычный браузер.
